# Архитектура VTS (Voice Task System)
*Архитектура информационной системы управления задачами на основе голосового ввода*

Автор: Токмаков Игорь 

Группа: РИС-22-2

## Описание системы
Разработчик создает интеллектуальную систему управления задачами, стремясь предоставить пользователям максимально быстрый и естественный способ планирования дел с помощью голоса и их автоматической синхронизации с календарем.

Пользователи: От десятков до сотен пользователей, ведущих активный личный и профессиональный образ жизни (фрилансеры, менеджеры, студенты).

Требования:
- Пользователи могут создавать задачи и устанавливать дедлайн для них с помощью голосовых команд на русском языке.
- Система автоматически распознает речь, извлекает суть задачи и время дедлайна, преобразуя их в структурированную запись.
- Система автоматически отправляет пользователю напоминание о задаче за заданное время до дедлайна.
- Пользователи могут выполнить одностороннюю синхронизацию созданных задач со своим Google Календарем, где задача становится событием.
- Система предоставляет простой и интуитивно понятный веб-интерфейс для просмотра и управления задачами.

Дополнительный контекст:
- Рынок персональных ассистентов и инструментов продуктивности быстро растет, и голосовые интерфейсы являются конкурентным преимуществом.
- Существующие решения (голосовые помощники, приложения-трекеры) предлагают либо базовый голосовой ввод без глубокой интеграции, либо мощные функции с ручным управлением, но не совмещают оба аспекта эффективно.
